# Incident Management Evidence-Based Assessment Tool

This repository contains the source code for the tool developed as part of a Master's thesis titled:

"A Unified Model for Information Security Incident Management: An Evidence-Based ISO/IEC 27035 Assessment Tool with AI-Driven Recommendations"

## Features
- Incident management maturity assessment
- Evidence-based evaluation
- AI-generated improvement recommendations
- Alignment with ISO/IEC 27035

## Technologies
Python
OpenAI API
Simulation model
